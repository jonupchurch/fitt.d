Wireframes · v1
Resume Analyzer & Job-Match Tool
Annotated low-fidelity wireframes and sitemap for a two-part web app: (1) upload a resume for analysis & advice, (2) paste a job description to compare against the resume. Guided wizard flow. Numbered pins map to notes; blue dashed boxes are responsive notes. Hand these to Claude in VS Code as the build spec.

ATS & formatting checks as a pass/fail list. Failures use the alert color and float to attention.
